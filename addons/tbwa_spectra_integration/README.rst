Integration module for TBWA Spectra platform connectivity.
Enables data synchronization between Odoo and Spectra systems.

Features
========

* Spectra API integration
* Data synchronization workflows
* Project status sync
* Resource allocation sync

Configuration
=============

1. Configure Spectra API credentials in System Parameters
2. Set up synchronization schedules
3. Map Odoo projects to Spectra projects
