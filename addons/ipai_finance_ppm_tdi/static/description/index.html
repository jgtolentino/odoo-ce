Import Types

Finance Team Members

Import finance personnel with proper roles and permissions. Roles: Finance Supervisor, Senior Finance Manager, Finance Director, Finance Assistant, Accounting Staff.

Month-End Closing Tasks

Import recurring tasks for month-end processes organized into 5 phases. Each task includes deadlines, responsibilities, and project assignments.

BIR Filing Calendar

Import Philippine Bureau of Internal Revenue filing schedules. Covers all major forms: 1601-C (monthly withholding), 2550Q (quarterly income tax), 1702-RT (annual), etc.
