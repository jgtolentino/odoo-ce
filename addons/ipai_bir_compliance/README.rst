BIR (Bureau of Internal Revenue) compliance tracking module for Philippine businesses.
Manages tax filing schedules, deadlines, and compliance status.

Features
========

* BIR form schedule management
* Filing deadline tracking
* Compliance status monitoring
* Integration with Finance PPM workflow

Configuration
=============

1. Navigate to Finance PPM > Compliance Calendar
2. Import or configure BIR filing schedules
3. Set up deadline notifications
