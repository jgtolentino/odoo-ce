This module provides **Planview-style PPM (Project Portfolio Management) dashboards** for Odoo CE 18.0.

**Key Features:**

* 4 Planview-inspired dashboard sections with ECharts visualization
* Comprehensive seed data for demo/testing purposes
* OCA-compliant module structure
* Apache ECharts integration for rich visualizations

**Dashboard Sections:**

1. **Project Planning & Management** - Dual gauge charts for project and budget health
2. **Portfolio Planning & Prioritization** - Treemap showing spend by strategy category
3. **Scenario Analysis** - Stacked area chart comparing planned vs actual vs earned value
4. **Intake & Demand Management** - Heatmap of project intake scores across metrics

**Demo Data:**

* 4 sample projects (Digital Banking Revamp, Core System Migration, Analytics Platform, Branch Kiosk)
* 5 financial periods (Q1-Q3 2025 Capex/Opex)
* 4 dependency nodes (critical path tracking)
* 3 intake requests (AI-Assisted Close, Legacy Decommission, Branch Kiosk)
* 2 AI insights (portfolio risks, high ROI opportunities)
* Kanban board with 3 columns and 4 cards
