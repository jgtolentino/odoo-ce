# Repository Structure  
Last updated: 2025-11-24 08:52 UTC  

This repository contains the **InsightPulse Odoo CE Platform** — a fully self-hosted deployment of Odoo Community Edition with OCA modules and custom `ipai_*` addons powering expense management, equipment booking, OCR automation, Supabase sync, and finance closing workflows.

The system is designed to:

- Replace **SAP Concur** for PH-focused expense and travel workflows.
- Replace **Cheqroom** for equipment catalog tracking, booking, and incident management.
- Run strictly on **Odoo CE + OCA + InsightPulse addons** — no Enterprise code paths.
- Block all **IAP usage, upsell UI references, and odoo.com cloud dependencies**.
- Operate exclusively under InsightPulse infrastructure (e.g., `erp.insightpulseai.net`).
